## SETUP:
```console
$ git clone https://github.com/AsiaMajor/core
$ cd core/
$ virtualenv env && source env/bin/activate
$ pip install -r requirements.txt
$ python server.py
```

## API endpoints:
* `/api/ping` with method `GET`
* `/api/preprocess` with method `POST`
